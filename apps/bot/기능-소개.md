# 환생 시뮬레이터 Discord 봇

실제 인구 분포 확률로 "다음 생"을 뽑고, 친구들과 함께 수집·대결하는 discord.js 봇입니다. 진행은 PostgreSQL에 저장되며, 뽑기 로직은 웹과 동일한 `packages/core`를 공유합니다.

## 슬래시 명령

| 명령 | 기능 |
|---|---|
| **/환생** | 실제 인구 분포 확률로 새 생을 뽑습니다. 일일 횟수 제한과 "공덕·업 계승" 버튼 제공. |
| **/여권** | 특정 생의 상세 정보(생번호 미지정 시 최신 생). `비공개` 옵션 지원. |
| **/덱** | 내가 모은 생 목록과 최고 기록. `공개` 옵션으로 채널에 공유. |
| **/명명** | 내 생에 원하는 이름을 붙입니다. |
| **/도감** | 서버가 **함께 모은** 국가 컬렉션(공동 수집형 진행). |
| **/배틀** | 다른 사람 덱과 **즉시 자동 판정** 대결. 상대의 수락이 필요 없습니다. |

## 동작 특징

- **공개가 확산 엔진** — 뽑기 결과는 채널에 공개. 단 "횟수 소진" 같은 거절 메시지는 본인에게만(ephemeral) 표시.
- **선택적 AI 요약** — 기본은 템플릿, LLM 설정 시 레어 생(상위 0.1%)에만 요약 생성하고 실패 시 템플릿으로 폴백.
- **stateless 버튼** — 상태를 `custom_id`에 인코딩해 재배포해도 과거 버튼이 그대로 동작.
- **상시 연결 프로세스** — 게이트웨이에 항상 연결된 장기 실행 봇. 슬래시 명령만 사용.
